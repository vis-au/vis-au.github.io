Protein Sequence Coverage Map

## Food science journal time ##

Authors

Soeren Drud-Heydary Nielsen Harith Rathish Hans-Jörg Schulz

Overview

Introduction

Related work: This plot is widely used in proteomics to provide an overview of the peptides associated to a protein. It is most commonly known in the literature as a "sequence coverage map" \cite. However, there is no formal publication on the visualization itself.

Structure of input data

Protein Sequences

Peptides

Order of stacking: For each amino acid "X", the longest peptide that starts with "X" is shown on top. From top to down, the length of the peptides decrease.

Color Scheme: We are using a categorical color scheme prescribed in Figure 4 of Qualititative Color Schemes in Paul Tol's notes.

Static mapping to bioactive functions: The bioactive functions map to the same colors as shown in the figure, regardless of their order in any input dataset.

Wrapping of lines: Peptides that span across amino acids near the edges of the plot wrap around and are shown on the axis below. The wrapped lines are indicted by arrow marks.

Length of signal peptide: This indicates the number of amino acids to hide from the beginning of the sequence.

Max axis length: This indicates the maximum number of amino acids shown per axis.

Export options: Currently we support exporting the plot as an SVG image, which contains the plot alone without the legend or the controls.
