About this Project

The concept for visualization toolchaining using a declarative visualization grammar as implemented in ReVize was published in "ReVize: A Library for Visualization Toolchaining with Vega-Lite" at STAG 2019.

When referring to the libraries and tools listed here, we kindly ask you to include the following citation:
