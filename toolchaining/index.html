Tools for Visualization Toolchaining

Developed by Marius Hogräfer, supervisor: Hans-Jörg Schulz

This page lists references to our open source software for Vega-Lite enabled visualization toolchaining.

ReVize A library for visualization toolchaining with Vega-Lite. ReModel A library for integrating in-situ toolchaining in visualization tools. ReVis A Vega-Lite enabled graphical editor for composite visualization. ReData A Vega-Lite enabled graphical editor for visualization dataflows. Color Brewer for Vega-Lite A Vega-Lite enabled fork of the popular ColorBrewer project. VisFlow with Vega-Lite A Vega-Lite enabled fork of the VisFlow project. KNIME Vega-Lite workflow A KNIME workflow for integrating Vega-Lite visualizations in data analysis.
